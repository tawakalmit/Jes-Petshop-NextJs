## 🧰 Installation

Clone this project

```sh
git clone https://github.com/tawakalmit/Jes-Petshop-NextJs.git
cd Jes-Petshop-NextJs
```

#### ⌛ Install

```sh
npm install
```

#### 🚀 Usage

```sh
npm run dev
```
